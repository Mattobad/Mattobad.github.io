House Price Prediction

Welcome to the House price prediction project. A regression problem to predict the House price given the characteristics of the house.


Project Description:

In this project, I've used feature selection techniques to select the important feature which later used for training the model. Using the technique 25 columns important columns(predictors) out of 81 were selected for predicting the target(house price). Features were selected particularly for randomforest regressor model. You can use the Prediction model tab from the menu to test the model.
